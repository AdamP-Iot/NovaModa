Popis projektu

Projekt NovaModa představuje fiktivní e-shop s urban/streetwear oblečením.

Cílem je demonstrovat schopnost vytvořit komponentovou knihovnu webového rozhraní pomocí HTML5, CSS (rozděleného do modulů) a JavaScriptu.

Web je kompletně responzivní, přístupný a využívá systematické stylování pomocí design tokens, komponentové struktury a utility tříd.


Struktura projektu

NovaModa/
│
├── css/
│ ├── reset.css # Resetuje výchozí styly prohlížeče
│ ├── tokens.css # Design tokens – barvy, spacing, typografie, stíny
│ ├── base.css # Základní typografie a elementární styly
│ ├── utility.css # Pomocné utility třídy (marginy, paddingy, flex)
│ ├── layout.css # Layout, gridy, sekce, responzivita
│ └── components.css # Komponenty (header, hero, karty, footer, tlačítka)
│ └── style.css # Import komponent v jedné třídě
│
├── js/
│ └── main.js # Javascript na propojení Json souboru s indexem.html
│
├── index.html # Hlavní stránka
